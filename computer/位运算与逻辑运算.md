## <<左移
```go
num := 1 << 2
fmt.Printf("%b", num)
```
1左移两位输出二进制100,，空缺的部分用0补上

## >>右移
```go
num := 10 >> 2
fmt.Printf("%b", num)
```
1010右移两位输出二进制10

## x ^ y[异或]
```go
num := 10 ^ 2
fmt.Printf("%b", num)

1010
^
  10
=
1000
```
操作的结果是如果某位不同则该位为1, 否则该位为0

## x | y[或]
```go
num := 10 | 2
fmt.Printf("%b", num)

1010
|
  10
=
1010
```

(2)奇数的二进制表示最后一位为1,所以可以通过位运算获取数之间的偶数
```go
func even(a int) (array []int) {
	for i := 0; i < a; i++ {
		if i&1 == 0 {
			array = append(array, i)
		}
	}
	return array
}
```

(3)互换两个变量的值，而不需要使用第三个变量做中间变量
```go
func swap(a, b int) (int, int) {
	a ^= b // 异或等于运算
	b ^= a
	a ^= b
	return a, b
}
```
